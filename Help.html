Guide d'utilisation

  1. Preparer l'itineraire de l'avion

    2. Lors du premier lancement du programme, il est necessaire de configurer l'itineraire. Si, celle-ci n'est pas faite avant de lancer la creation d'un itineraire, un message d'erreur apparaitra. Sinon, la configuration est chargée/
    La configuration est necessaire pour obtenir une carte issue de l'api Google Static Map
    Plus d'info sur Google static Map

    En ce qui concerne le calcul des breakpoints tout est fait en local par le programme.

    1. Configuration

      1. Configuration de la vue

        • Taille de l'image

          La taille de l'image peut varier entre
          200x200 et 640x640 pixels

        • Facteur de zoom

          Le facteur de zoom peut varier entre 1 et 19.
          19 etant la limite fixee par Google.
          Un bon reglage permet d'observer correctement la zone a survoler.a

      2. Configuration de l'espace entre les breakpoints

        3. Le nombre de breakpoint est un paramètre important.
        En effet, plus il y a de breakpoins, plus l'url a envoyer à Google
        est grande. Google a fixe une limite de 2048 caracteres
        Si l'image ne s'affiche pas, augmentez l'espace inter-breakpoints

      3. Configuration de la zone de survol

        4. Pour configurer la zone de survol, il faut renseigner quatres points du globe
        par leurs latitude et longitude. La forme de la zone de survole doit etre rectangulaire.

      4. Configuration du lieu d'enregistrement

        5. A chaque fois que vous creer un itineraire, les donnees sont automatiquement enregistrees
        dans le dossier renseigne dans le champs de texte.

    2. Visualisation

      3. Une fois l'adresse envoyee a Google, celui-ci retourne
      une image qui sera affichee dans l'application. Celle ci est centree
      sur la zone de survole et montre les breakpoints des virages et le chemin a
      parcourir.

    3. Partage

      4. Les donnees issues de la creation de l'itineraire sont facilement utilisables.
      En effet, un fichier texte comportant toutes les coordonnées est automatiquement
      genere pour le pilote automatique.
      Un fichier kml est egalement genere pour faciliter le partage et offrir une visualisation
      plus confortable des donnees
      En effet, le fichier kml peut facilement est ouvert dans Google mpa ou Google earth
      Plus d'info sur le format KML

  2. Construire le champs georeference

    3. Preparer les donnees apres le vol

    Une fois le vol termine et le Rapberry Pi recuperer. Oter la carte memoire.
    Puis, sur le pc ou le logiciel est installe copier les dossiers GPS et PHOTO
    normallement present sur la carte sd dans le dossier /home/pi

    Construire le panorama

    Une fois les donnees copiees, il suffit de lancer le programme et via le menu
    Fichier puis creer. Une fenetre de selection de dossier apparait.
    Choisissez le dossier comportant les dossiers GPS et PHOTO et validez.
    Enfin, il suffit d'attendre la fin du traitement.

  3. Utilise le panorama

    1. Options disponibles

      • Bouger l'image

        L'image peut etre deplace par drag and drop avec la souris.

      • Zoomer

        Differentes options sont disponibles pour agrandir ou reduire la
        taille de l'image. Soit avec la molette de souris, soit via le menu contextuel
        que vous pouvez ouvrir via un clic droit sur l'image.

      • Centreer

    2. Definition de zone ou marqueur

      3. Un double clic ou l'option Ajouter du menu contextuel vous permet d'ajouter une pin si il n'y a pas de pin dejo
      ajoutee a cette endroit. Sinon, la precedente pin est supprimee.

      Des lignes relient automatiquement deux pins successives.
      Les ligne sont automatiquenent supprimees si une pin qui lui est reliee est supprimee.

      Toute les actions peuvent etre annulee via le menu Edition et l'option annuler ou annuler tout.

    3. Sauvegarder et reutiliser votre travail

      4. Plusieurs options sont disponibles :
      Sauvegarde sous forme d'image via l'option Sauvegarder Image du menu Fichier, le projet est fige et n'est plus exploitable
      par le logiciel.

      Sauvegarde sous forme de zone image en faisant un drag avec le clic droit, idem qu'auparavant mais perment l'enregistrement
      d'une zone precise du panorama.

      Sauvegarde des positions via le menu Fichier, qui permet de pouvoir reutiliser
      le projet via l'optio Charger du memu Fichier.
